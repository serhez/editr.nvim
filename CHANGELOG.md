# Changelog

## 0.1.0 - 2026-05-21

Initial public tester release.

- Add Neovim-side context detection for sessions launched by `editr`.
- Add remote-aware Snacks file and grep pickers that open paths in the local
  mirror.
- Add canola/oil entry points, canola selection routing, and hydration handling
  for ignored remote-only files.
- Add router helpers so existing local mappings can route to editr-aware tools
  during remote sessions.
